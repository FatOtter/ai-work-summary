## 🚀 快速开始

### 方法 1：直接打开
直接在浏览器中打开 `index.html` 文件即可查看。

### 方法 2：使用本地服务器
```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用提供的脚本
chmod +x serve.sh
./serve.sh
```

然后在浏览器中访问：`http://localhost:8000`
